## User Interface

### View Types

Scientific Project Manager provides four main view types:

| View Type | Best For | Features |
|-----------|----------|----------|
| **Tree** | Browsing lists, comparing data | Sortable, filterable, exportable |
| **Form** | Detailed editing, relationships | Tabs, chatter, attachments |
| **Kanban** | Visual workflow, status tracking | Drag-and-drop, grouping |
| **Calendar** | Timeline planning, scheduling | Time-based visualization |

Learn more in [Views & Navigation](views-navigation.md).

### Navigation Tips

!!! tip "Keyboard Shortcuts"
    - `Alt + C`: Create new record
    - `Alt + S`: Save record
    - `/` or `S`: Open search
    - `Ctrl + K`: Command palette

!!! tip "Filters"
    Use the search bar to:
    - Filter by any field
    - Group by categories
    - Save custom filters
    - Share filters with team

## Integration Features

### Odoo Chatter

All core modules integrate with Odoo's chatter for:

- **Messages**: Communication within context
- **Activities**: Task scheduling and reminders
- **Followers**: Notification subscriptions
- **Email Integration**: Send/receive emails

### Activity Tracking

Changes are automatically tracked:

- Status changes
- Assignment modifications
- Date updates
- Field edits

View history in the chatter section.
